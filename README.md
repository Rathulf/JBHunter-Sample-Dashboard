JBHUNTER | Job Search Management Dashboard
  -JBHUNTER is a professional web-based dashboard designed to help users track, organize, and manage their job applications in one centralized location. This project demonstrates my ability to take a complex administrative task and build a structured, user-friendly digital solution.

Purpose & Vision
  -Searching for work can be disorganized and overwhelming. JBHUNTER was created to provide a clean, visual interface for tracking application statuses, company details, and deadlines, reflecting my core strengths in organization and attention to detail.

Key Features
  -Centralized Application Tracking: Designed a structured layout to keep all job-related data in a single view.
  -Professional UI Design: Built using HTML and CSS to ensure a clean, modern aesthetic that is easy to navigate.
  -Strategic Categorization: Features distinct sections for different stages of the hiring process, showcasing my ability to categorize and prioritize information.
  -User-Centric Layout: Developed with a focus on efficiency, ensuring that the most critical information is visible at a glance.

Technical Toolkit
  -HTML5: Structured the dashboard for clear information hierarchy.
  -CSS3: Applied custom styling to create a responsive and professional user experience.
  -Product Management: Managed the project from initial concept to refined code.
